A One-Project Man

After learning to code a few years back, I only recently set off to explore becoming an engineer. This project quickly became a fun collision of the personal and professional as I set off to learn about back-end engineering while automating something annoying and making it more accessible to my friends.

This scoreboard first lived in an Excel spreadsheet that I had to manually update each week. A screenshot of the updated scoreboard would be shared via Slack and that was that, but the process was begging for improvement, and the prospect of hosting a scoreboard online seemed like the best solution.
